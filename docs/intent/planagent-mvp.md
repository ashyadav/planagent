# Intent: PlanAgent MVP

- **Outcome:** A working plan→execute agent where you can watch tool calls stream into the UI in real time
- **User:** You, learning how agents work by building one
- **Why now:** Hands-on understanding of the ReAct loop and tool-calling
- **Success:** Type a task, agent plans it, SearXNG tool calls stream in step by step in the UI
- **Constraint:** MVP only — one tool (SearXNG via local instance), no shell, no file I/O
- **Out of scope:** Brave Search, shell tool, file I/O, approval toggle, path-traversal guards — all post-MVP
